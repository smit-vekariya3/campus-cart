# Campus Cart 🛒
A full-stack peer-to-peer marketplace designed for college students to buy and sell used items securely.

## ✨ Features
- **User Auth:** Secure login/register with JWT and bcrypt.
- **Product Management:** Upload items with multiple images using Multer.
- **Wishlist:** Save favorite items to your personal dashboard.
- **Responsive UI:** Fully optimized for mobile and desktop using Tailwind CSS v4.

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite, Axios
- **Backend:** Node.js, Express.js, MongoDB
- **Styling:** Tailwind CSS

## 🚦 Getting Started
1. Clone the repo.
2. Run `npm install` in both `client` and `server` folders.
3. Add a `.env` file to the server with `MONGO_URI` and `JWT_SECRET`.
4. Use `npm run dev` for frontend and `node server.js` for backend.
